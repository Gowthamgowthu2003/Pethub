PetHub is a full-stack J2EE web application developed using Servlets, JSP, JDBC, and MySQL. The system enables users to manage pet adoption listings with complete CRUD operations while following the MVC architecture pattern.

Project Features:
- User registration and login
- Add, update, and delete pet listings
- View available pets for adoption
- Admin management functionality
- Database connectivity using JDBC
- MVC-based architecture implementation

Technologies Used:
- Core Java
- Servlets and JSP
- JDBC
- MySQL
- Apache Tomcat
- HTML and CSS

Project Structure:
PetHubProject/
- src/
- WebContent/
- README.md

Setup Instructions:
1. Import the project into Eclipse IDE.
2. Configure Apache Tomcat Server.
3. Create a MySQL database.
4. Update database credentials in the DB connection file.
5. Run the project on the server.
